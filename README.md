# AI-Driven Logistics & Delivery Management System

## Overview
This project is a system designed to optimize last-mile delivery using AI-based decision support tools.

## Roles
- Customer
- Driver
- Admin

## Core Features
- Authentication
- Order Management
- Driver Assignment
- Real-time Tracking
- Notifications
- Admin Dashboard

## Intelligent Features
- Adaptive Route Optimization
- Smart ETA Prediction
- OCR-Based Delivery Verification
- Driver Performance Analysis

## Tech Stack
- Flutter
- Node.js
- PostgreSQL
- Python (AI models)
